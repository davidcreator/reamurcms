## [2.0.0] - 2025-XX-XX

### 🚀 Added

#### 🎓 Online Courses & Virtual Classrooms *(New Module!)*
- Course creation and curriculum management
- Lesson editor with support for video, text, and quizzes
- Student enrollment and progress tracking
- Virtual classroom management with access controls
- Live session scheduling and recording integration
- Certificate generation upon course completion
- Instructor dashboard and analytics

#### 🏠 Landing Pages *(New Module!)*
- Visual builder for high-converting landing pages
- Customizable sections, CTAs, and forms
- A/B testing support and conversion tracking
- Integration with CMS content and e-commerce products

#### Core
- Improved admin panel performance
- Extended REST API with new endpoints for courses and classrooms
- Redis and Memcached cache support
- Improved multi-language system

### 🔄 Changed
- Refactored MVCL architecture for better scalability
- Improved responsive design across all modules
- Updated PHP minimum requirement to 8.0
- Optimized database queries across modules

### 🐛 Fixed
- Fixed session handling in multi-language environments
- Fixed e-commerce checkout on mobile browsers
- Fixed image upload validation
